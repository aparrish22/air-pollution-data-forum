This is a readme to help you run virtual environment and run the server locally.

Open up cmd prompt

Use "cd" command to navigate through directory.

Go into Project 3 directory and run the cmd "django\forum\Scripts\activate.bat"
 in cmd shell to start virtual environment

After starting venv, go into "Project 3\mysite"

and this is where you will be able to run server with "py manage.py runserver"

After running server, go to 127.0.0.1:8000 on your preferred web browser to open up the website.

To go to admin webpage, simply go to 127.0.0.1:8000/admin/


UPDATE - refer to README.txt in mysite folder.
